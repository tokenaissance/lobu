Lobu deploys sandboxed [OpenClaw](https://github.com/openclaw/openclaw) agents on demand for every user and channel. OpenClaw is an open-source agent runtime — Lobu wraps it in multi-tenant infrastructure so every user gets their own isolated agent from a single deploy.

## Quick start

```bash
npx @lobu/cli init my-agent
cd my-agent && npx @lobu/cli dev -d
```

See the [CLI Reference](/reference/cli/) for all available commands.

After deployment, connect your messaging platforms (Telegram, Slack, Discord, WhatsApp, Teams) via the admin page at `{PUBLIC_GATEWAY_URL}/agents` — no platform-specific env vars needed.

Managed access is currently free.

Notes:

- Pricing will be based on compute machine usage in the future.
- LLM keys belong to you (bring your own provider keys).
- OpenClaw workers scale to zero when idle and wake when you send a message.

## Self-hosted

Self-hosted gives you full control of infrastructure and runtime policy.

Main differences from managed:

- You run the infrastructure (Docker or Kubernetes).
- You manage ops, networking policy, and upgrades.
- You pay infra costs directly in your own environment.
